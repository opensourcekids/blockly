Welcome to PyBlocks

PyBlocks uses colorful blocks to write programs instead of typing.

  • Blocks
    • Blocks are used to write a program.
    • Blocks are stored in the menu.
  • Menu
    • The gray menu on the left.
    • Some menus have menus inside.
    • Labels group the blocks by category.
    • Clicking a label shows the blocks inside.
    • Click a block to place it onto the workspace.
    {#
    #}
  • Workspace
    • This is the large white area to the right of the menu.
    • All PyBlocks programming happens here.
    {#
    #}
  • Python Code Area
    • Underneath this panel is the Python Code Area.
    • It is accessed by clicking on the Code tab.
    • Python code is generated here whenever the workspace changes.
    {#
    #}
  • Output Area
    • Underneath this panel is the Ouput Area.
    • It is accessed by clicking on the Output tab.
    • The program runs when the Output tab is clicked.
    • Only printable output shows up here.
    {#
    #}
  • Canvas
    • Pops up when a graphical program like Turtles runs.
    • The ouput to the program is shown here.
    • To exit, click the close button or the shaded area.
