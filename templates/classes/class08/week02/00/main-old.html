Python and Math


  • For all simple math opertations, Pyblocks uses


  • Python uses + for addition and - for subtraction.
  • Other operations use symbols different from written ones:

      • * for Multiplication

      / for Division

      ** for Powers

  • There is also an operator you may not have heard of called modulo; it uses a %. This is the same as the remainder in long division.
  • Examples:

      • 4 % 2 = 0

      3 % 2 = 1

      17 % 3 = 2

      19 % 4 = 3
