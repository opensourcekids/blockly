Welcome to SnapPy Code

            
SnapPy Code uses colorful blocks to write programs instead of typing them.
            

            
    
                
  • Blocks
    • 
                
      
                    
    • Blocks are used to write a program.
      • 
                    
    • Blocks are stored in the menu panel.
      • 
                
    
                
  • Menu
    • 
                
      
                    
    • The blue menu panel in on the left side of the screen.
      • 
                    
    • The menu panel contains a list of other menus.
      • 
                    
    • These named menus group blocks by category.
      • 
                    
    • Clicking a menu name shows the blocks inside.
      • 
                    
    • Click a block to place it onto the workspace.
      • 
                    
    • Some menus contain submenus.
      • 

                
    
                {#                
    #}
                
  • Workspace
    • 
                
      
                    
    • The large white area to the right of the menu is the workspace.
      • 
                    
    • All SnapPy Code programming happens here.
      • 
                    
    • There is a trash can on the lower right of the workspace.
      • 
                    
    • Drag blocks to the trash can to remove them from the workspace.
      • 
                
    
                {#                
    #}
                
  • Python Code Area
    • 
                
      
                    
    • Underneath this panel is the Python Code Area.
      • 
                    
    • It is accessed by clicking on the Code tab.
      • 
                    
    • Python code is generated here whenever the workspace changes.
      • 
                
    
                {#                
    #}
                
  • Output Area
    • 
                
      
                    
    • Underneath this panel is the Ouput Area.
      • 
                    
    • It is accessed by clicking on the Output tab.
      • 
                    
    • The program runs when the Output tab is clicked.
      • 
                    
    • Only printable output shows up here.
      • 
                
    
                {#                
    #}
                
  • Canvas
    • 
                
      
                    
    • Pops up when a graphical program like Turtles runs.
      • 
                    
    • The ouput to the program is shown here.
      • 
                    
    • To exit, click the close button or the shaded area.
      •
